Til opsætning af docker containers
 1: Åbn commandprompt
 2: Sæt directory til der hvor docker-compose.yml filen befinder sig ved at bruge 'cd' kommandoen efterfulgt af stien til hvor compose filen befinder sig
  Eksempel: cd C:\Users\Rasmus Høy Schmidt\source\repos\UCN\1. semester\Eksamen\Georgia-Tech-Library
 3: Kør kommandoen 'docker compose up'
 4: Inde i browseren kan man tilgå swagger interfacet via 'http://localhost:5119/swagger'
